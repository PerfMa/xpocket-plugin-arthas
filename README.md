### Arthas Plugin For XPocket
#### 简介
Java诊断工具Arthas的XPocket插件实现

#### 操作指南
使用
``` shell
use arthas@ALIBABA
```
进入Arthas插件域
然后使用
``` shell
help
```
或者
``` shell
system.help arthas@ALIBABA
```
来获取详细帮助信息

更多操作以及介绍请参考[官方介绍](https://arthas.aliyun.com/)
